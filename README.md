
## 1. Project Overview
This project is part of the Portfolio Assessment for the University of Bolton. We are exploring the implementation of **Akaunting**, an open-source accounting software, to solve financial management problems for small and medium-sized enterprises (SMEs) in Greece.

## 2. Business Problem
Greek SMEs often face high licensing costs for proprietary ERP systems and lack of localization for Greek accounting standards. Our solution focuses on:
* Reducing operational costs via Open-Source software.
* Improving localization (Greek translation and terminology).
* Ensuring data sovereignty using a self-hosted solution.
