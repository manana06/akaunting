# Contributing to the Akaunting Greek Edition Project

As part of our University Portfolio Assessment, this project follows specific contribution guidelines to ensure high-quality localization and business alignment for the Greek market.

## How We Contribute
Our team focuses on three main areas:
1. **Localization (L10n):** Translating and refining accounting terminology in `resources/lang/el-GR` to meet Greek legislative standards.
2. **Business Documentation:** Updating README and project docs to reflect SME business problems.
3. **Quality Assurance:** Testing the interface for Greek character rendering and UI consistency.

## Team Contribution Workflow
1. **Fork the Repository:** Each member works on their own branch or local environment.
2. **Commit Standards:** Use clear commit messages (e.g., "Refactor: Updated invoice labels").
3. **Peer Review:** Team members review each other's language files before final submission.

## Code of Conduct
We maintain a professional and collaborative environment, respecting the open-source values of the original Akaunting community.
